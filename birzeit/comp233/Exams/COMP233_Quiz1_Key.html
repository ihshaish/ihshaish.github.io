Question One [45 pts]: Logical equivalence, tautology, and an inference rule

Q1.a [15 pts]

Simplify the following logical expression to an equivalent expression:

¬(p ∧ ¬q) ∧ (p ∨ q)

Solution Apply equivalence laws step by step. The target equivalent expression is q.

#FormLaw used
0¬(p ∧ ¬q) ∧ (p ∨ q)given
1≡ (¬p ∨ ¬(¬q)) ∧ (p ∨ q)De Morgan's law on ¬(p ∧ ¬q)
2≡ (¬p ∨ q) ∧ (p ∨ q)double negation: ¬(¬q) ≡ q
3≡ q ∨ (¬p ∧ p)distributive law (factor out q)
4≡ q ∨ cnegation law: ¬p ∧ p ≡ c
5≡ qidentity law: q ∨ c ≡ q

Final ¬(p ∧ ¬q) ∧ (p ∨ q) ≡ q.

Marking, Q1.a [15 pts, 3 pts per correct step]
PtsCriterion
3Step 1: De Morgan applied correctly to ¬(p ∧ ¬q).
3Step 2: double negation applied to ¬(¬q).
3Step 3: distributive law to factor out q.
3Step 4: negation law identifying ¬p ∧ p ≡ c.
3Step 5: identity law reaching q.

Notes Accept equivalent orderings of the laws. Accept NOT, ~, or ∼ in place of ¬. If the student names a different law that yields the same intermediate form correctly, award full marks for that step.

Q1.b [25 pts]

Show that the following expression is a tautology by proving the logical equivalence using a truth table:

(p ∧ (p → q)) → q  ≡  T

Solution Build the truth table column by column. Four rows for two variables p, q:

pqp → qp ∧ (p → q)(p ∧ (p → q)) → q
TTTTT
TFFFT
FTTFT
FFTFT

The final column is T in every row, so the expression evaluates to true for every assignment of truth values to p and q.

Verdict (p ∧ (p → q)) → q ≡ T. The expression is a tautology.
Marking, Q1.b [25 pts]
PtsCriterion
3Correct column headers (the five columns above).
3Correctly computed intermediate columns: p → q and p ∧ (p → q).
164 pts per correct row (4 rows). Lose 1 pt per wrong cell.
3Correct conclusion stating the expression is a tautology and equivalent to T.

Notes Award 0 pts if the truth table is missing entirely. If only the final column is given without intermediate columns, cap at 8 pts.

Q1.c [5 pts]

Which Rule of Inference is represented by the tautology in part (b)? Write its formal argument form.

Rule Modus Ponens. The tautology (p ∧ (p → q)) → q says: if both p and p → q are true, then q is true.

Form

p p → q ∴ q
Marking, Q1.c [5 pts]
PtsCriterion
2Correctly naming Modus Ponens.
3Correct formal argument form: two premises (p and p → q) and conclusion ∴ q. All three lines must be present.

Notes Accept equivalent notations for the conclusion (∴ q, ⊢ q, "therefore q"). 0 pts if the rule name is wrong.

Question Two [55 pts]: Conditional variations, predicates & quantifiers, short answers

Q2.a [20 pts]

Write the contrapositive, converse, inverse, and negation of the following statement:

"If Sara passes Discrete Math, then she will register for Data Structures and she will take Algorithms."

Setup Let p = "Sara passes Discrete Math", q = "Sara registers for Data Structures", r = "Sara takes Algorithms". The original statement is p → (q ∧ r).

Contrapositive
¬(q ∧ r) → ¬p  ≡  (¬q ∨ ¬r) → ¬p
"If Sara does not register for Data Structures or does not take Algorithms, then she does not pass Discrete Math."
Converse
(q ∧ r) → p
"If Sara registers for Data Structures and takes Algorithms, then she passes Discrete Math."
Inverse
¬p → ¬(q ∧ r)  ≡  ¬p → (¬q ∨ ¬r)
"If Sara does not pass Discrete Math, then she does not register for Data Structures or does not take Algorithms."
Negation
¬(p → (q ∧ r))  ≡  p ∧ (¬q ∨ ¬r)
"Sara passes Discrete Math, and she does not register for Data Structures or does not take Algorithms."

Important The negation is not a conditional. It is a conjunction: the hypothesis holds and the conclusion fails.

Marking, Q2.a [20 pts, 5 pts per variation]
PtsCriterion (apply to each of the four variations)
3Correct symbolic form for the variation.
2Correct verbal sentence in English.

Notes De Morgan must be applied to ¬(q ∧ r) in the contrapositive, inverse, and negation. Deduct 2 pts overall if missing throughout. Accept ¬q ∨ ¬r or ¬r ∨ ¬q. 0 pts for the negation if it is written as a conditional.

Q2.b [20 pts]

Predicates on the domain D = {2, 3, 5, 6, 8}:

E(x) = "x is even";  P(x) = "x is prime".

For each of (i)–(ii), write the statement formally with quantifiers and predicates, then give its truth value with justification. For the Rewrite, give the English sentence and the truth value with justification.

(i) Every element in D is even.  [5 pts]
(ii) There exists an element in D that is both even and prime.  [5 pts]

Rewrite the following formal statement as a complete English sentence and give its truth value with justification:  [10 pts]

∀x ∈ D, (E(x) → P(x))

(i) Formal: ∀x ∈ D, E(x).   Truth value: False.   Justification: x = 3 ∈ D is not even. (Also x = 5.)

(ii) Formal: ∃x ∈ D, (E(x) ∧ P(x)).   Truth value: True.   Justification: x = 2 ∈ D is even and prime.

Rewrite English: "For every element x in D, if x is even then x is prime."

Truth value: False.   Justification: x = 6 ∈ D is even (6 = 2 · 3) but is not prime; the hypothesis holds and the conclusion fails. (x = 8 is another counterexample.)

Marking, Q2.b [20 pts]
PtsCriterion
2(i) Correct formal statement ∀x ∈ D, E(x).
1(i) Correct truth value (False).
2(i) Justification with a specific counterexample (x = 3 or x = 5).
2(ii) Correct formal statement ∃x ∈ D, (E(x) ∧ P(x)).
1(ii) Correct truth value (True).
2(ii) Justification with a specific witness (x = 2).
4Rewrite: correct English sentence (universal conditional read aloud).
2Rewrite: correct truth value (False).
4Rewrite: justification with a specific counterexample where the hypothesis holds but the conclusion fails (x = 6 or x = 8).

Notes A bare truth value with no specific witness/counterexample loses the justification marks. Generic claims ("there is one") without a named element score 0 for that step.

Q2.c [15 pts, 3 pts each]

Provide the direct logical equivalent or operation for each of the following.

(1) Write the negation of p → q.
(2) Which variation of p → q is logically equivalent to its Converse?
(3) Given p ↔ q, write it as a conjunction of two conditional statements.
(4) Write the formal argument form of Modus Tollens (using p and q).
(5) Let D = {1, 2, 3, 4} and let P(x) = "x is a factor of 8." Write the truth set of P(x).

(1) ¬(p → q) ≡ p ∧ ¬q. Reading: "p is true and q is false." The negation of a conditional is not itself a conditional.

(2) Inverse: ¬p → ¬q ≡ q → p. The converse and inverse are contrapositives of each other and are therefore logically equivalent.

(3) p ↔ q ≡ (p → q) ∧ (q → p).

(4) Modus Tollens:

p → q ¬q ∴ ¬p

Reading: "If p then q. Not q. Therefore not p."

(5) Truth set: {1, 2, 4}. The factors of 8 are 1, 2, 4, 8; intersect with D = {1, 2, 3, 4}. 8 is a factor of 8 but 8 ∉ D, so it is excluded.

Marking, Q2.c [15 pts, 3 pts each]
PtsCriterion
3(1) Correct answer p ∧ ¬q. Accept ¬(p → q) written as-is but only 1 pt if not simplified.
3(2) Correctly naming the Inverse. 1 pt if only the symbolic form ¬p → ¬q is given without naming it.
3(3) Correct answer (p → q) ∧ (q → p). Accept equivalent ordering. 0 pts if disjunction or wrong connective is used.
3(4) Modus Tollens: 1 pt for premise p → q, 1 pt for premise ¬q, 1 pt for conclusion ∴ ¬p. All three lines must appear for full marks.
3(5) Correct truth set {1, 2, 4}. Deduct 1 pt if 8 is included (domain restriction missed). Deduct 1 pt per other wrong or missing element.

General Marking Principles

  • For proof-style and equivalence-step questions, marking is sequential: students lose individual marks for missing or incorrect steps, not the whole question.
  • Accept either negation glyph throughout: ¬, ~, or ∼.
  • In Q2.a, the negation must not be expressed as a conditional. De Morgan must be applied to ¬(q ∧ r) in the contrapositive, inverse, and negation.
  • In Q2.b, justifications without a specific witness or counterexample score 0 for the justification step.
  • In Q2.c, item 4 (Modus Tollens) requires all three lines (two premises and the conclusion) for full marks.
  • If a student names a different law in Q1.a that yields the same intermediate form correctly, award full marks for that step.
